## Project Overview

- Collect, clean, and analyze financial and investment data
- Learn and apply Python, Pandas, Matplotlib/Seaborn
- Practice visualization with Tableau or Power BI
- Explore basic investment metrics and indicators

## Folder Structure

- `/data` : Raw datasets (CSV, Excel, etc.) and processed datasets for analysis
- `/notebooks` : Jupyter notebooks with experiments and analysis
- `/scripts` : Python scripts for reusable code
- `/visualizations` : Graphs, charts, and Tableau/Power BI outputs
- `/reports` : PDF reports or analysis summaries from projects
- `/references` : Reference materials, tutorials, articles, and investment/finance resources
